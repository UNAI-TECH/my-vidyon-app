## ğŸ”‘ Key Concepts

### 1. **TranslationProvider**
- Wraps entire application
- Provides global translation state
- Manages localStorage persistence

### 2. **useTranslation Hook**
- Access translations anywhere
- Get/set current language
- Type-safe access to all keys

### 3. **LanguageSelector**
- Reusable UI component
- Dropdown with all languages
- Can be placed anywhere

### 4. **Translation Files**
- One file per language
- All must have same keys
- Type-checked by TypeScript

### 5. **Type Safety**
- `TranslationKeys` type ensures consistency
- Autocomplete in IDE
- Compile-time errors for missing keys
